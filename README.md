# Stuckify - Personal Portfolio Website

Um site de portfólio pessoal moderno, vibrante e responsivo desenvolvido com HTML5 e CSS3. O projeto simula a página inicial de uma criadora de conteúdo digital (Youtuber e Blogger), apresentando uma interface rica em elementos visuais abstratos, tipografia marcante e uma disposição fluida de componentes de navegação.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do cabeçalho, seção principal (Hero) e links sociais.
* CSS3: Estilização com gradientes lineares, posicionamento de elementos flutuantes e formas circulares.
* Flexbox / CSS Grid: Alinhamento da barra de navegação superior e organização dos elementos de texto e imagem.

## Funcionalidades e Técnicas Aplicadas

* Cabeçalho de Navegação Completo: Menu superior contendo links para Home, Latest, Category, Services e Contact, integrado a uma barra de pesquisa minimalista.
* Efeito de Texto Digitado (Simulado): Destaque na apresentação profissional utilizando cores contrastantes para enfatizar as ocupações ("Youtuber, Blogger").
* Formas Geométricas de Fundo: Uso de círculos com gradientes suaves em tons pastel e multicoloridos posicionados estrategicamente atrás dos textos e da imagem principal para criar profundidade.
* Botões de Ação com Gradiente: Botões "About Me" e "Follow Me" personalizados com transições de cores modernas.
* Barra de Redes Sociais Lateral: Ícones de mídias sociais (Facebook, Twitter, LinkedIn, Pinterest) alinhados de forma sutil no canto inferior direito da interface.
